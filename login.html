About the Academic Monitoring System

The Academic Monitoring System (AMS) is a centralized platform created to improve academic management and monitoring within Talisay City College.

The system allows administrators to manage departments, approve accounts, configure sections, assign teachers, and monitor system activity efficiently.

Students can monitor their academic records, grades, evaluations, and requirements while teachers can manage academic performance and student progress securely.

Key Features

Account Approval System

All student and teacher accounts require administrator approval before gaining access.

Academic Monitoring

Monitor evaluations, grades, academic standing, and student performance in real time.

Department & Section Management

Administrators can organize departments, majors, year levels, sections, and subject assignments.

Secure Access

User authentication and monitoring help maintain system security and accountability.
